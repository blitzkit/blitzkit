# BlitzKit 1.30.0

Thank you to [GonnaHetzMe](https://github.com/karelpak1) for your contributions this update.

## Features

- Added an equalizer mode for Tankopedia and Compare to preview what tank stats will look like in the upcoming Equalizer game mode
  - This feature is in Beta and the stats are subject to change
  - Thanks to WoTB News for sourcing data well ahead of the release of the gamemode
  - Added a toggle in Tankopedia, right beneath Provisions
  - Added button under the shell selector in the Tank Sandbox
  - Added another toggle in the Compare table to enable equalizer for all tanks

## Technical Features

- [GonnaHetzMe] Enabling dev mode now displays the tank's non-numerical id
- [GonnaHetzMe] The dev id and the new dev name are now copyable on click

## Fixes

- [GonnaHetzMe] Tech trees that start at tiers higher than tier 1 like the Blitz Nation ones now properly render their trees in Tankopedia
- [GonnaHetzMe] The tank pitch/yaw inputs in the Tank Sandbox now initialize with the correct values instead of 0
- [GonnaHetzMe] Large Repair Kits once again use the correct modern icon
