## Run the bot

### Install dependencies

```bash
yarn
```

### Create `.env`

```env
GH_TOKEN={YOUR_GITHUB_TOKEN_HERE}
DISCORD_TOKEN={YOUR_DISCORD_TOKE_HERE}
```

### Run the command

```bash
yarn dev:bot
```
